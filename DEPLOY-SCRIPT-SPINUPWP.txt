# ⚡ COMANDO EXACTO PARA SPINUPWP - DEPLOY SCRIPT

## Pega EXACTAMENTE esto en el campo "Deploy Script (optional)" de SpinupWP

```bash
#!/bin/bash
cd /var/www/micomicona
exec ./spinupwp-deploy-hook.sh
```

---

## Después marca esto:
✅ **"Run this script after every site deployment"**

---

## ¿Qué hace?

1. Cambia al directorio del proyecto
2. Ejecuta el script `spinupwp-deploy-hook.sh` que:
   - Compila el fetcher en Go
   - Procesa los 31 feeds RSS
   - Compila el sitio con Hugo
   - Genera el HTML estático
   - Nginx lo sirve automáticamente
